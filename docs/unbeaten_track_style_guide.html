02 — Typography

Single typeface throughout — Inter via Google Fonts. Three weights only: 400 (regular), 500 (medium), 600 (semibold). No serif, no display font.

/* Google Fonts import */

03 — Buttons

Buttons use display: inline-flex — never <button> elements in Blade templates where widget styles may interfere. Border-radius 7px on standard buttons, 6px on small.

12 — Notes for Cursor / developer handoff

Buttons: Use display: inline-flex not <button> in Blade templates. Browser and admin template default button styles override background colours on standard button elements.
Photos: One 3:2 master image per POI stored in the database. Never request multiple crops. All display sizes handled by background-size: cover; background-position: center via CSS.
Day selector: Show as pill tabs for 1–4 days. Switch to a <select> dropdown for 5+ days to prevent tab overflow. Implement conditionally in Alpine.js: x-show="journey.days.length <= 4".
Map: MapLibre GL JS replaces all SVG map placeholders in the real build. The SVG mockups show intended marker styles, popup design and legend layout only.
Font loading: Load Inter via Google Fonts in the HTML <head>. Use display=swap to prevent FOUT. The Limitless admin template may already include Inter — check before adding a duplicate import.
Narration block: The left-bordered quote block uses border-left: 3px solid var(--color-mid-green) with border-radius: 0 on the left side only. Do not apply border-radius to single-sided borders.
Colour palette: The coral/orange from the original logo has been retired. Do not introduce it. The full palette is defined in the CSS custom properties block above — use only those tokens.
